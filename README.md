# 💥 Custom 404 Page

> A stylish, responsive “404 Not Found” error page template built with HTML, CSS  — ideal for personal websites, portfolios, or any static site needing a polished fallback.

---


## 🧠 About the Project  
This project provides a fully designed custom “404 – Page Not Found” template that replaces the default generic error page with something more branded, engaging, and user-friendly.  
When visitors land on a non-existent URL, this template:
- Displays a clear “404” title and message  
- Matches your site’s visual theme or branding  
- Includes a call-to-action (e.g., a link back to the homepage)  
- Ensures a good user experience even when an error occurs  

---

## ✨ Features  
- ✅ Fully responsive layout (desktop, tablet & mobile)  
- ✅ Easily customizable styles — update colors, fonts, animations  
- ✅ Optionally includes interactive elements (animations, fun message)  
- ✅ Lightweight and fast — minimal/no dependencies  
- ✅ Easy to integrate into any static site or GitHub Pages  

---

## 🧱 Built With  
- **HTML5** — structure and semantic markup  
- **CSS3** (Flexbox/Grid) — layout, styling, responsiveness

### Installation  
```bash
# 1. Clone the repository
git clone https://github.com/iamyoussefezzat/custom-404page.git  
cd custom-404page

# 2. Open locally or deploy:
#    If it's pure static: simply open index.html in your browser  
open index.html  

#    If deploying on GitHub Pages:  
#    Push to your `main` branch and enable GitHub Pages.
```
